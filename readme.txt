This plugin gives you a dropdown to quickly change which post you are editing in the admin panel.

== Description ==

A very lightweight plugin that adds a metabox onto the add/edit screen for any post type (besides attachment, revision, and nav_menu_item) that allows you to quickly switch between other posts in the post type without having to go back to view all posts.

== Installation ==

1. Upload directory to your plugins folder
2. Go to Plugins in your WordPress Admin Panel
3. Activate Quick Post Change plugin
